重装debian

昨天早上，毫无征兆地，我的debian 7的x windows竟然崩了，开始怀疑只是gdm坏了，重装了下不行，在LOG文件里找了找也没找到有用的东西（也没很认真地找）。电脑还要用，重装了独显驱动也无济于事后，我开始打算怎么重装系统了。

最后还是换回原来的双系统吧，单单debian系统会遇到些尴尬的情况，比如这次那机动车理论考的光盘（里面是个win安装包，如果是网页多好，也不用安装，打开就能做题）。

备份，格盘，分区。

先装windows7，安装，激活，更新驱动，装常用软件，搞定！

接着拿出我的debian 7装光盘，开始安装debian 7

开始检测的时候竟然缺失固件，都快忘了这茬，赶紧拿U盘去官网下了驱动，解压放在根目录。接着就按部就班，分区的时候睁大眼睛别看错就好。

完成后弹出光盘，重启，GRUB，选择debian 7启动，选择gnome classics（咱不需要3D桌面）

增加软件源，下面是我的sources.list

更新软件列表

卸载了原有的输入法，安装了ibus（拼音+五笔）

安装了latex

显卡驱动，分辨率没问题，就不装了。用着没问题就可以了。

sudo命令

系统已经有了sudo命令，只是还没配置好

编辑/etc/sudoers,添加这句

usename ALL=(ALL:ALL) ALL

安装好c/c++编程环境

$sudo apt-get install build-essential

安装vim和emacs，从源里安装就可以了，配置文件是先前备份好的。拷回来就可以了。其中，emacs的一个插件-autex，需要重新在电脑上编译安装，才可以使用。 安装git

$sudo apt-get install git $sudo apt-get install git-core

安装jekyll

$sudo gem insall jekyll

安装okular,icedove,chmsee,基本就够了。 还有修改键盘映射（用vim或者emacs的都懂的），gnome桌面设置有修改键盘布局的选项。
