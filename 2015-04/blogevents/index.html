博客大事记

2015年4月28日，我的博客从github page搬到搬瓦工vps。

博客在 github page 上安家2年有余，杂乱的记了一些事、一些心情、一些折腾笔记。整个博客没有主题可言， 完全随着性子写，想放什么放什么，实在是因为生活学习上没有过深的感悟和探究，草草记录点，草草放在这儿。

博客将来的内容还是杂揉，但涉及linux等技术性的东西将会大大减少，源因为我发现先前很多文章完全可以做成个人wiki的内容，这也是菜单栏出现wiki的原因。很多重复的，罗列类的东西将会被放在wiki里。

用jekyll来生成静态博客，lighttpd做web服务器，访问速度和在 github page 上差不多。在 github page 上有个好处，内容不怕丢，每次更新都有备份。 数据备份很重要，打算五一把博客内容的备份问题解决，到时会删掉github上的truehyp.github.com库，那原地址就不会再跳转到houye.xyz了。

因此，以后的博客地址就只是http://houye.xyz

一年11.99美元的vps，挺耐用的。建了几个账号给朋友玩，搭了博客服务器，还可折腾下shadowsocks。
