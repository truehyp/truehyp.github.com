标量

标量(scalar)：单个事物，例如数字，字符串

数字

  • 数字在内部都以双精度浮点数储存
  • 直接量(literal)，直接写在源码中的数据
  • 操作符，%取模，**乘幂

字符串

  • use utf8; 将源代码以utf8处理
  • 单引号字符串，反斜线转义只用于\\和\'
  • 双引号字符串，反斜线转义功能同C语言，\n,\t等

操作符

  • 连接操作符，.（句点)
  • 字符串重复操作符，x（小写的x）

perl会完成字符串和数字的转换工作，你采用的是什么操作符，perl就会对操作数对相应的转换

标量变量

  • 储存单个标量值的变量，如$name，$是取单个东西/取标量的意思
  • 操作符 = += *= .=(字符串连接符号)
  • 输出 print
  • 字符串中的标量变量内插 ，要在双引号里，print "a is $a";如果$a未初始化，则用空字符串替代。

比较操作符

  • 字符串：数字
  • ==:eq
  • !=:ne
  • <:lt
  • >:gt
  • <=:le
  • >=:ge

if判断语句

  • if(){} 一定要加花括号

布尔值

  • 数字：o为假，其余为真
  • 字符串：空字符串为假（字符串'0'也为假），其余为真
  • 不是数字和字符串：转换成数字或字符串再判断

去掉字符串末尾的换行符

  • chomp()

while 控制语句

  • while(){} 一定要加花括号

undef 值

  • 未赋值的标量变量，为数字时是0，为字符串时是空字符串

判断是否为undef 值

  • defined()
  • 创建undef 值 $aaa = undef

列表和数组

列表和数组代表复数。列表指标量的有序集合，数组指储存列表的变量

数组索引值

  • $#name 为最后一个元素的索引值。有个更简单的记法,-1表示数组最后一个元素的索引值

列表直接量

  • (1, 2, 3)
  • (1..100)

qw简写

  • quoted word/quoted by whitespace 以空白符分割，加上引号，组成单词列表
  • qw(a b c d)

引用整个数组

  • @name
  • @rocks = qw(bedrock slate lava)

pop 和push 操作符

  • pop(@name)
  • push(@name)
  • 数组为空，返回undef

shift 和unshift 操作符

  • 与pop push类似，但操作的是数组开头

splice 操作符

  • splice @array, 1, 2, qw(www) 删除array 下标1开始的2个元素，并用"www"替换。后2个参数可选。

字符串中数组内插

  • print "a is @rocks"

foreach 控制结构

  • foreach $var (qw( a b c)){} var 取值，遍历列表，进行迭代
  • 默认变量 $_
  • foreach (1..8){} 忽略控制变量，默认以$_为控制变量

reverse 操作符

  • 返回逆序的列表，不修改传进来的变量

sort 操作符

  • 返回排序后的列表，不修改传进来的变量

each 操作符

  • 返回数组下一个元素对应的索引和值
  • ($index, $value)=each @rocks

标量上下文和列表上下文

  • chomp(@lines = <STDIN>) 读入所有行，换行符除外

子程序

定义

  • sub marine {}

调用

  • &marine

默认数组变量

  • @_ 是程序的私有变量
  • 第一个变量 $_[ 0] 第二个变量 $_ [ 1]

my 操作符

  • my($m, $n) 创建私有变量

一个例子

$maximum = &max(3, 5, 10, 4, 6)
sub max {
     my（$max_so_far) = shift @_;
     foreach (@_){
	  if ($_ > $max_so_far){
	       $max_so_far = $_;
	  }
	  $max_so_far;
     }

use strict 编译命令

  • 编译时采用更为严谨的方式，先用my 声明，再使用

return 操作符

  • 从子程序中返回某个值
  • 持久性私有变量
  • state $n = 0;

输入输出

文件操作

  • open CONFIG, 'dino';
  • open CONFIG, '<dino';
  • open BEDROCK, '>fred';
  • open LOG， ‘>>logfile';
  • open CONFIG, '<:encoding(UTF-8)', 'dino'; 以UTF-8读取文件，也可以以此写入文件
  • open BEDROCK, '<:crlf', $file_name; 以dos风格读取文件（\r\n）

二进制

  • binmode STDOUT
  • binmode STDERR
  • binmode STDOUT, ':encoding(UTF-8)';

关闭文件句柄

  • close BEDROCK

处理致命错误

if (!ipen LOG, '>>', 'logfile'）{
     die " cannot create logfile: $!";
}
  • $! 为默认错误信息，如 permission denied 等
  • die 会终止程序并返回非零的退出码

自动检测致命错误

  • use autodie;

改变默认的文件输出句柄

  • select BEDROCK;
  • print "aaaaa"

输出内容会送到该文件句柄

  • say 操作符
  • 类似println 输出自动换行
  • 标量变量中的文件句柄
  • 将文件句柄存在标量变量中

哈希

访问哈希元素

  • $hash{$some_ksy}
  • $hash{'fred'}

访问整个哈希

  • %family_name

胖箭头

my %last_name = (
'fred' => 'flintstone',

keys 和values 函数

  • kyes 返回哈希的键列表
  • values 返回哈希的值列表

each 函数

  • 以列表形式返回键值对

exists 函数

  • if (exists $books{'dino'}{} 检查哈希表中是否存在某个值

delete 函数

  • delete $books{$person}; 删除哈希表指定键和对应的值

正则表达式

匹配Unicode属性

  • if (\p{Space){}
  • if (\p{Digit}){}
  • if (\P{Space}){} 大写的P，匹配模式以外的文本
  • if ((.)\1){} \1 表示对应顺序的捕获组，此匹配连续的2个同样的字符
  • if ((.)\g{1}11){} 用方向引用\g{1} 消除歧义。使用负数表示相对引用。

用正则表达式进行匹配

用m// 进行匹配，一般的匹配都是忽略了m。也可以用{} %%定界

模式匹配修饰符

  • if (/yes/i){} /i 大小写都匹配
  • /s 匹配任意字符，包括换行符
  • /x 允许加入空白符号，可以在模式中随意加入空白符，使他更易读理解

锚点

  • \A 匹配字符串绝对开头
  • \z 匹配字符串绝对末尾 \z 允许后面出现换行符
  • ^ $ 行首和行尾

匹配多行文本

  • 对perl 来说，这是一个大字符串，即使里面有好多换行符，但对人来说它是多行的。
  • m 是将字符串作为多行处理，s是将字符串作为单行处理，如果是s在字符串中出现的\n就相当于普通字符。
  • /fred$/m 找出行末的fred字符串，匹配fred和随后的换行符

单词锚点

  • \b 匹配单词边界
  • \B 匹配非单词边界
  • \bhunt\b 会匹配hunt ,但不会匹配hunting，hunter
  • \bhunt\B 会匹配hunter,hunting，但不会匹配hunt

=~绑定操作符

  • if ($some_other =~ \brub){} 让右边的模式匹配来匹配左边的字符串，而不是默认匹配的$_

捕获变量

  • 捕获到的变量保存在字符串，可以用$1,$2,这样的形式来表示它们
  • 捕获变量的存续期
  • 能存活到下次成功匹配为止。

不捕获模式

  • if ((?:bronto)?saurus){} 只分组不进行捕获

命名捕获

  • if ($name = ~ m/(?<name1>\w+)(?:and?or){} 可以用用$+（name1）引用
  • 自动捕获变量
  • 三个捕获变量
  • 通用量词

花括号

  • {n,m} 匹配n到m次

用正则表达式处理文本

  • 用s///进行替换
  • 用s///g 进行全局替换
  • \s+ 匹配空白符

无损替换

  • 加/r 修饰符。s///原来是操作完后返回成功替换的次数，加/r后会保留原字符串变量中的值不变，而把替换结果作为替换操作的返回值返回

大小写转换

  • \U 会将其后的所有字符转换成大写
  • \L 会将其后的所有字符转换成小写
  • \u \l 只会影响其后的第一个字符

split 操作符

  • my @fields = split separator, $string;
  • 例如 my @fields = split \s+, $string; 以空白字符分隔字段

join 函数

  • 将片段合并为一个字符串
  • my $x = join ";", 4, 6, 8, 10, 12;

非贪婪模式

  • 后面加？
  • 一次更新多个文件

其他控制结构

unless 控制结构

  • 除非执行条件为真，否则就执行里面的代码。相当于独立的else子句

until 控制结构

  • 在条件为假时重复执行，或者说重复执行直到条件为真

裸块

  • 没有关键字的代码块

elsif 子句

for 控制结构

for 其实和foreach 是等价的

last 操作符

  • 相等于break

next 操作符

  • 类似于continue

redo 操作符

  • 从新执行这次的迭代

带标签的块

  • 可以从内层对外层的循环块进行控制 用last next redo 进行跳转 
LINE: while (<>) {
     foreach (split) {
	  last LINE if /_END_/;
}
}
