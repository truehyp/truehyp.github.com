ffmpeg

转换音频
ffmpeg -i input.wav output.mp3
视频转换
ffmpeg -i input.mpeg -s 500*500 output.flv
从视频中截取图片 默认一秒截取25张
ffmpeg -i input.mpg image%d.jpg
每秒截取1张
ffmpeg -i input.mpg -r 1 image%d.jpg
从第十秒开始，每秒截取25张图片，截取5秒时长的图片
ffmpeg -i input.mpg -r 25 -ss 00:00:10 -t 00:00:05 image%05d.png
从视频中采集音频
ffmpeg -i input.avi -f mp3 output.mp3 ffmpeg -i input.avi -vn output.mp3 只输出音频
用图片制作视频
ffmpeg -f image2 -i image%d.jpg ./output.mpg
截取视频
ffmpeg -i input.flv -ss 00:00:10 -t 00:00:20 output.flv
例子1
ffmpeg -i 理学院毕业晚会2013.mpg -ab 128 -acodec libmp3lame -ac 2 -r 29.97 -b 512 -ss 00:00:00 -t 00:05:54 li3.mpg
例子2
ffmpeg -ss 01:00:00 -i input_file_h264.mp4 -vcodec copy -acodec copy -t 00:06:00 output_file.mp4
